# Duplicate_Question_Pair_Quora_NLP_Project
* The Quora Question Pairs dataset, hosted on Kaggle, is a machine learning competition dataset containing over 400,000 pairs of questions, aiming to identify if they are semantic duplicates. 
* It was released to improve answer consolidation and user experience by detecting redundant queries, with labels created by human experts.

* Dataset Link: https://www.kaggle.com/c/quora-question-pairs
